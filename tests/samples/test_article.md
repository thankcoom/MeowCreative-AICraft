## Claude Code 寫手系統的核心功能

### 1. 智能內容分析

系統使用先進的 NLP 技術分析原始文章，自動提取：
- 核心主題和關鍵字
- 文章結構和段落組織
- 語氣和寫作風格
- SEO 優化建議

這讓你可以快速了解一篇文章的優缺點，並知道如何改進。

### 2. 自動 SEO 優化

我們的 **SEO Analyzer** 提供 0-100 分的評分系統，涵蓋：

- **關鍵字優化** (25 分)：密度、位置、分布
- **內容質量** (25 分)：字數、可讀性、結構
- **技術 SEO** (25 分)：標題、meta、連結
- **使用者體驗** (25 分)：視覺元素、掃描友好度

每個維度都提供具體的改進建議和預期影響，讓你知道「優化這個可以提升 15% 的 CTR」。

### 3. 一鍵發布到 WordPress

不用再手動複製貼上！**WordPress Publisher** 可以：
- 自動轉換 Markdown 為 HTML
- 上傳特色圖片
- 設定分類和標籤
- 配置 SEO meta 標籤
- 排程發布時間

只需要一個命令，就能完成所有發布流程。

## 如何開始使用？

### 步驟 1：安裝系統

```bash
git clone https://github.com/your-repo/ai-writer-system
cd ai-writer-system
./scripts/setup_skills.sh
```

### 步驟 2：配置 WordPress 憑證

編輯 `.claude/config/wordpress-credentials.yaml`：

```yaml
site_url: "https://your-blog.com"
username: "your-username"
application_password: "your-app-password"
```

### 步驟 3：開始寫作

只需要告訴 Claude Code 你想做什麼：

> "請將這篇文章改寫成我的風格，優化 SEO，然後發布到 WordPress"

系統會自動完成所有工作！
