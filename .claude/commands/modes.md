# 顯示可用模式 (v2.6.0)

顯示 AI 寫手系統所有可用的執行模式。v2.6.0 新增 Search Everywhere 搜尋優化模式。

---

## 執行動作

展示以下執行模式選項：

```
================================================================
喵哩文創 AI 寫手系統 v2.6.0 - 可用模式
================================================================

⚡ quick (快速模式)
   只執行 Critical Phases，快速完成
   預估時間: ~5 分鐘
   適用場景: 時間緊迫、簡單內容

📝 standard (標準模式) [預設]
   執行 Critical + Important Phases
   預估時間: ~15 分鐘
   適用場景: 一般部落格文章

🔥 full (完整模式)
   執行所有 Phases (包含 Optional)
   預估時間: ~30 分鐘
   適用場景: 重要內容、完整行銷素材

📱 publish (發布模式)
   標準模式 + WordPress 發布 + 多平台
   預估時間: ~20 分鐘
   適用場景: 需要自動發布到平台

📖 story (故事模式)
   強調 Storyteller Agent
   預估時間: ~20 分鐘
   適用場景: 個人經驗、故事敘事

🎯 seo (SEO 模式)
   強調 SEO Optimizer + Research
   預估時間: ~20 分鐘
   適用場景: 搜尋引擎優化需求

🌐 search (Search Everywhere 模式) 🆕 v2.6.0
   完整 AISO 優化 (SEO + GEO + LLMO + AEO + VSO + CRO)
   預估時間: ~25 分鐘
   適用場景: 最大化跨平台搜尋能見度

🤖 llmo (LLMO 模式) 🆕 v2.6.0
   強調 AI 搜尋優化 (ChatGPT/Perplexity 引用)
   預估時間: ~20 分鐘
   適用場景: 希望被 AI 系統引用

🎙️ voice (語音搜尋模式) 🆕 v2.6.0
   強調 VSO 語音搜尋優化
   預估時間: ~15 分鐘
   適用場景: 語音助理回答優化

================================================================
快捷別名:
  q → quick      se → search
  s → standard   l  → llmo
  f → full       v  → voice
  p → publish
================================================================

2025 搜尋趨勢:
  • AI 搜尋流量成長 1,200%
  • AI 用戶轉換率是傳統的 4.4 倍
  • 65% 搜尋以零點擊結束
================================================================
```

## 使用方式

透過 `/write` 命令使用模式：

```
/write https://example.com --quick
/write https://example.com --mode full
```

或使用互動式選擇（預設行為）。
